御泥坊APP售后问题汇总

1.什么时候发货？优先发货指的是什么？

付款成功后我们会在48小时候尽快发出,正常情况下24小时内能发货成功。
在APP内下单的用户，享受各大分仓优先发货的特权，保证发货快人一步

2.多久可以收到货？

根据不同的地址,一般会在2-5天到货。

3. 发什么快递？

为保证您能够尽快收到货,系统根据您的收获地址选取最快捷的快递进行配送.目前有顺丰,圆通,汇通。

4.为什么我的物流信息不显示？

由于目前物流接口存在不稳定的情况,可能会有物流信息显示之后或者不显示的情况 亲也可以在相关物流的官网进行查询,获取最新的物流信息。

5.什么是15天无理由退货？

我们承诺15天无理由退货!您无论出现什么问题,在付款当日起15天内提出申请,切退还货物不影响二次销售。我们都可以退款
